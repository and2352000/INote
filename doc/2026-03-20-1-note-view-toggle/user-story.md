## 背景

目前分類頁面的筆記僅有列表（list）一種呈現方式。使用者希望能在列表與卡片（block/grid）之間切換，以便在不同情境下用最適合的方式瀏覽筆記。

## 使用者故事

As a 筆記使用者
I want to 在 Topbar 切換筆記的顯示模式（列表 / 卡片）
So that 我可以依偏好選擇最適合的瀏覽方式

## 驗收條件

- [ ] Topbar 新增顯示模式切換按鈕（list / block icon toggle）
- [ ] 列表模式（List）：維持現有樣式，一行一篇，顯示標題、日期、描述、tags
- [ ] 卡片模式（Block）：以格狀卡片呈現，每張卡片顯示標題、日期、描述、tags
- [ ] 切換為全站統一設定，所有分類頁面共用同一顯示模式
- [ ] 使用者的選擇透過 localStorage 記憶，重新整理或換頁後維持上次選擇
- [ ] 預設顯示模式為列表（List）

## 備註（選填）

- 首頁（分類列表頁）不受影響，僅分類內的文章列表頁適用
- 響應式設計：卡片模式在小螢幕應自動調整欄數
